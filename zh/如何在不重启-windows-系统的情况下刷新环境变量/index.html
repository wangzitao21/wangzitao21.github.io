如何在不重启 Windows 系统的情况下刷新环境变量

2024/5/19 · 汪子涛 | 语言:
  • 🌐 Switch to English

在 Windows 系统中，有时添加一些可执行文件的环境变量，如 ffmpeg、modflow 等，通常需要重启系统才能生效。

然而重启往往需要关闭许多正在进行的任务，很麻烦。

以下步骤可以在不重启系统的情况下刷新环境变量：

  1. 将需要添加的程序的路径添加到环境变量中。
  2. 打开 CMD，输入 echo %Path% 查看当前的环境变量。
  3. 输入 set PATH=C:。
  4. 关闭 CMD 并重新打开，再次输入 echo %Path% 会发现环境变量已更新。
