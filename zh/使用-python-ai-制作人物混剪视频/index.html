使用 Python+AI 制作人物混剪视频

参考人工剪辑会被 AI 替代吗？。

剪辑思路#

  1. 收集需要混剪的人物素材，如做莫妮卡·贝鲁奇的混剪视频，需要收集她的电影，短片等等。将所有的视频放在第一个文件夹 Material。

视频文件名中不要包含 .。因文件基于 . 修改后缀，文件名中包含 . 会报错。

  1. 在网上找几张莫妮卡·贝鲁奇的照片放在第二个文件夹 Portrait 里。这些参考图片用来作为从视频中识别的参照对象。

需要注意，参考图片的亮度不要太暗，太暗的视频识别准确率略低。不要使用太多种不同尺寸图片，容易造成识别错误。一个简单的办法是直接从现有视频里截取几张。

  1. 将 Material 里的所有视频在各自的转场处分割，产生大量的短片，放到 Videos 文件夹。由 split 函数实现。
  2. 将 Videos 中所有的短片第一帧均导出为图片，并放到 Images 文件夹。由 get_thumbnail_from_video 函数实现。
  3. 使用人脸识别，将 Images 里每张图片依次与 Portrait 中的参考图片进行对比。一旦被识别为同一个人，就将 Videos 里对应的视频移动到 Output 文件夹。由 face_con 函数实现。
  4. 在 Premiere Pro 里安装 Beatedit 插件。插入音乐，自动打点匹配节奏。

具体步骤#

  1. 创建并进入名为 mashup 的环境。
1

conda create -n mashup python=3.10 && conda activate mashup
  1. 安装模块。此外，FFmpeg 要事先配置在环境变量里。
1

pip install ffmpy face_recognition scenedetect -i https://pypi.tuna.tsinghua.edu.cn/simple
  1. 收集图片和视频。并创建上述所有被提到的文件夹。
1

mkdir Material Portrait Videos Images Output

import os
import shutil
import ffmpy
import face_recognition
import subprocess as sp

def split(materialPath, videosPath):
    """
    将所有素材在转场处分割并移动到 Videos 文件夹
    """
    materialList = os.listdir(path=materialPath)
    for material in materialList:
        material = os.path.join(materialPath, material)
        cmd = "scenedetect --input %s detect-content list-scenes split-video --output %s" % (material, videosPath)
        print(cmd)
        p = sp.Popen(cmd, shell=True)
        p.wait()

def get_images_from_videos(videosPath, imagesPath):
    """
    将 Videos 文件夹里每个视频的第一帧导出为图片, 并移动到 Images 文件夹
    """
    videosList = os.listdir(path=videosPath) # 获取视频文件夹中的所有文件名，存储在一个列表中
    for video in videosList:
        image = video.replace(".mp4", ".png")
        ff = ffmpy.FFmpeg(
            inputs={os.path.join(videosPath, video): None},
            outputs={os.path.join(imagesPath, image): ['-ss', '00:0:01.000', '-vframes', '1']}
        )
        ff.run()

def facial_recognition(imagesPath, portraitPath, videosPath, outputPath):
    """
    根据图片对比结果, 将的确是同一个人的视频移动到 Output 文件夹
    """
    portraitsList = os.listdir(path=portraitPath)
    portraitEncoding = []
    for portrait in portraitsList:
        portrait = face_recognition.load_image_file(os.path.join(portraitPath, portrait))
        portraitEncoding.append(face_recognition.face_encodings(portrait)[0])
    
    imagesList = os.listdir(path=imagesPath)
    for image in imagesList:
	    imageName = image # 供后面使用
        image = face_recognition.load_image_file(os.path.join(imagesPath, image))
        imageEncoding = face_recognition.face_encodings(image) # 如果没有面部信息返回的是空表

        if len(imageEncoding) > 0:
            results = face_recognition.compare_faces(portraitEncoding, imageEncoding[0]) # portraitEncoding 是一个列表，可以传多个图片对比
            if results[0] == True:
                print("图片中包含目标人物")
                video = imageName.split(".")[0] + ".mp4"
                output = shutil.move(os.path.join(videosPath, video), outputPath)
                print(f"已将 {video} 移动到：{output}")
            else:
                print("图片中不包含目标人物")

def create_folder(foldersList):
    """
    检测文件夹是否存在, 不存在则新建一个
    """
    for folder in foldersList:
        if not os.path.exists(folder):
            os.mkdir(folder)

if __name__ == '__main__':
    materialPath = r"./Material"
    videosPath = r"./Videos"
    imagesPath = r"./Images"
    portraitPath = r"./Portrait"
    outputPath = r"./Output"

    create_folder([videosPath, imagesPath, outputPath])
    split(materialPath=materialPath, videosPath=videosPath)
    get_images_from_videos(videosPath=videosPath, imagesPath=imagesPath)
    facial_recognition(videosPath=videosPath, imagesPath=imagesPath, portraitPath=portraitPath, outputPath=outputPath)
