贝叶斯优化是一种全局优化方法，用于优化黑盒函数。

  1. 定义目标函数，如准确率或损失值。
  2. 使用先验模型，如高斯过程，用于对给定的数据点进行目标函数代理建模。
  3. 根据先验模型和已观察到的数据点，选择采集函数。常用的采集函数包括期望改进（Expected Improvement, EI）和最大概率改进（Probability of Improvement, PI）。例如，EI的公式如下：

$$ EI(x) = E[max(f(x)-f(x^+), 0)] $$

其中，f(x) 是目标函数，f(x+) 是目前找到的最佳函数值。

  1. 迭代优化。在每次迭代中，我们会通过采集函数选择一个采样点，然后在这个采样点处评估目标函数，得到一个新的观测值。这个新的观测值会被加入到我们的数据集中。然后，我们会使用这个更新后的数据集来更新我们的代理模型（也就是高斯过程模型）。这样，我们的代理模型就能够更好地反映目标函数的形状。
  2. 重复上述步骤。

![[贝叶斯优化和 MCMC_00.png]]

“查询点"就是我们在下一次迭代中要评估的点256。在每次迭代中，我们都会根据采集函数来选择一个新的查询点，并计算其对应的目标函数值256。然后，我们会使用这个新的数据点来更新我们的先验模型256。在下一次迭代中，我们会再次选择一个新的查询点，并重复这个过程

2 先验代理模型#

高斯过程在贝叶斯优化中被用作代理模型。

在贝叶斯优化中，我们的目标是优化一个昂贵的、难以评估的目标函数。

由于直接优化这个函数可能非常困难，我们使用高斯过程来建立这个函数的代理模型。

这个代理模型可以帮助我们理解目标函数的行为，并预测我们还未观察到的点的函数值。

因此，你可以把高斯过程看作是在这里代理了目标函数。

3 采集函数#

期望改进（Expected Improvement, EI）是一种常用的采集函数。

EI的主要思想是，在每次迭代中，选择那些能够最大化期望改进量的点进行评估。

EI的公式如下：

EI(x)=E[max(f(x)−f(x+),0)]

其中，f(x) 是目标函数，f(x+) 是当前找到的最佳函数值。

在每次迭代中，我们都会计算所有候选点的 EI 值，并选择 EI 值最大的点作为下一个查询点。

这样，我们就可以在探索（寻找尚未评估的区域）和利用（寻找已知信息）之间取得平衡，从而有效地找到全局最优解

蒙特卡洛#

蒙特卡洛方法是一种随机模拟方法，通过大量随机样本的统计结果来估计问题的解。 在这个例子中，我们随机生成点，并统计落在圆内的点的比例，从而估计圆的面积。

  1. 在一边长为1的正方形内绘制半径为0.5的圆，圆心位于正方形中心。
  2. 正方形面积为1，圆面积为 π·r 2 = π/4。
  3. 落在圆内点m的数量与圆面积成正比。
  4. 模拟点总数 n 与正方形的面积成正比。
  5. 由3、4得到圆内点的比例 m/n 近似等于圆的面积与正方形面积的比值，即

m/n≈(π/4)/1=π/4

直接上代码

import random
import matplotlib.pyplot as plt

def monte_carlo_pi(n, s=10):
    """
    使用蒙特卡洛方法估算圆周率，并可选地进行可视化.

    Args:
        n: 模拟点的数量.
        visualize: 是否进行可视化.

    Returns:
        圆周率的估计值.
    """
    in_circle_count = 0
    x_in, y_in = [], []  # 圆内的点
    x_out, y_out = [], []  # 圆外的点

    for _ in range(n):
        x = random.uniform(0, 1)
        y = random.uniform(0, 1) 
        distance = (x - 0.5)**2 + (y - 0.5)**2 
        if distance <= 0.25:
            in_circle_count += 1
            x_in.append(x)
            y_in.append(y)
        else:
            x_out.append(x)
            y_out.append(y)

    pi = 4 * in_circle_count / n

    # 创建图形
    fig, ax = plt.subplots(figsize=(2,2))

    # 绘制正方形
    ax.set_aspect('equal')
    ax.plot([0, 1, 1, 0, 0], [0, 0, 1, 1, 0], color='black')

    # 绘制圆
    circle = plt.Circle((0.5, 0.5), 0.5, color='blue', fill=False)
    ax.add_patch(circle)

    # 绘制点
    ax.scatter(x_in, y_in, color='red', s=s, label='Inside')
    ax.scatter(x_out, y_out, color='green', s=s, label='Outside')
    ax.set_xticklabels([])
    ax.set_yticklabels([])
    ax.set_xticks([])
    ax.set_yticks([])
    for spine in ax.spines.values():
        spine.set_visible(False)

    # ax.legend()
    plt.text(0.8, 1.05, f"$\pi={pi}$")
    plt.title("Pi (n={})".format(n))
    plt.show()

    return pi

# 设置模拟点的数量
# n = 1000

# 计算圆周率的估计值并进行可视化
pi_estimate = monte_carlo_pi(n=10, s=10)
pi_estimate = monte_carlo_pi(n=100, s=10)
pi_estimate = monte_carlo_pi(n=1000, s=5)
pi_estimate = monte_carlo_pi(n=10000, s=0.1)
pi_estimate = monte_carlo_pi(n=100000, s=0.1)
