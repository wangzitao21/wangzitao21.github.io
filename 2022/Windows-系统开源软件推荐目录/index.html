Windows 系统开源软件推荐目录

Posted on 2022-06-24 Edited on 2022-10-16 Views: Views: Valine:
Symbols count in article: 9.6k Reading time ≈ 9 mins.

作为软件收集爱好者，我近些年来陆陆续续收集整理和试用了一些软件。因此制定了本软件目录清单。

最近更新时间: 2022 年 9 月 19 日。

本软件目录清单收录的宗旨:

  • 含有链接的通常都是开源免费软件，如果不是开源的，那至少也是免费的。收费和部分收费软件会做专门说明。
  • 链接都是该软件的官网或者 Github 或其他平台的下载页面。
  • 拒绝和抵制一切含广告、诱导安装和全家桶类型的垃圾软件。例如在压缩软件目录中，类似 2345，好压等软件绝不会被本清单中收录。
  • 本软件目录清单将持续更新。
  • 有个别软件只支持 Linux，MacOS 或移动端，和本清单题目不符，但数量很少，可以忽略。

1 网页浏览器及其插件

浏览器作为互联网之窗，首先必须杜绝 360 浏览器、QQ 浏览器和搜狗浏览器等等诸如此类的套壳浏览器。主要推荐以下三种:

  • Chrome: 谷歌浏览器。
  • Microsoft Edge: 在国内大环境下，Edge 应该是最好用的浏览器了。
  • Firefox: 火狐浏览器。

Edge 兼容 Chrome 扩展，因此可以无缝衔接，我主要使用以下插件:

  • SingleFile: 把网页保存为单一的 *.Html 文件。
  • Fatkun: 批量下载图片。
  • Infinity: 为浏览器定制新标签页，与其相似的还有 青柠起始页、Momentum 等等。
  • Tampermonkey: 油猴脚本，可在 Greasy Frok 下载脚本。
  • 划词翻译: 用途如其名，类似的插件有很多。
  • Listen 1: 音乐播放器。
  • Kiwix: 搭配下载离线的 *.zim 文件浏览维基百科内容。
  • EPUB Reader: Epub 格式的阅读器，类似的插件有很多。
  • Save to Zotero: Zotero 附带的浏览器扩展 (见 Zotero)。
  • Proxy SwitchyOmega: 代理管理。
  • Bilibili-Evolved: B 站增强 Tampermonkey 脚本。

还有很多有用有趣的插件可以见 插件英雄榜。此外，很多插件已经不再需要，如解除浏览器禁止复制的插件，生成网站二维码的插件，兼容 Internet Explorer 模式的插件等等，这些功能 Edge 本身均已经可以胜任。

2 文档与文本

包括各类办公、打印或笔记类的文档或文本类的程序。

2.1 Office

  • Microsoft Office: 收费，但难以割舍。
  • Open Office / Libre Office: Microsoft Office 的开源替代品，但我不习惯使用。

Office 也有一些常用插件:

  • Office Tab Enterprise: 多个 Office 文件呈类浏览器的标签形式，收费，鸡肋，不用。
  • EasyShu: 在 Excel 中绘制各类图件，收费，鸡肋，不用。
  • Office Timeline Pro: 制作 PPT 时间轴，收费，鸡肋，不用。
  • MathType: 公式编辑器，属于收费软件，目前没有找到合格的开源替代品。像 AxMath 也是收费。其实用惯了感觉 Office 自带的公式编辑器也愈发顺眼了。
  • OneKeyTools: PPT 插件。
  • 小恐龙公文排版助手: 排版插件。

2.2 PDF

  • Acrobat Reader DC: Adobe 的 PDF 阅读与编辑器，收费，不过功能很全。
  • SumatraPDF: 能打开包括 PDF、ePub、MOBI 等多种格式，打开速度快，但界面和功能简陋。
  • PDFsam: 可进行 PDF 的裁剪，合并，提取等操作，小巧简陋。
  • XODO PDF: PDF 处理的在线工具。
  • jPdf Tweak: PDF 拆分等工具。
  • PDFtk: PDF 处理工具。
  • PDF 24: 目前感觉是最好用的开源免费的 PDF 处理工具。

2.3 ePub

  • Calibre: 电子书库，功能强大，自带阅读器和 ePub/azw3 编辑器。
  • Sigil: ePub 编辑器。
  • Koodo Reader: ePub 阅读器。
  • Foliate: ePub 阅读器，只支持 Linux。

其他如 ePubBuilder 编辑器，Alfa eBooks Manager 电子书库，Icecream Ebook Reader, EPUB File Reader, Digital Editions 等 Epub 浏览器，均收费且不见得好用。

还有一些 Calibre 和 Sigil 的插件:

  • ePub3-itizer: Sigil 插件，用于将 ePub 2 转成 ePub 3。
  • DeDRM_tools: Calibre 插件，用于去除 DRM 加密。

2.4 Notes

  • Joplin: 类似印象笔记，可借助第三方工具 (如坚果云) 多平台同步，还可以安装网页插件 Joplin Web Cipper 实现剪藏功能。
  • BoostNote: 与 Joplin 类似。
  • Zettlr: 自称适合学术和文学创作。
  • VNote: Markdown 编辑器。
  • Notable: Markdown 编辑器。
  • MacDown: macOS 的 Markdown 编辑器。
  • Moeditor: Markdown 编辑器。
  • Abricotine: Markdown 编辑器。
  • Haroopad: Markdown 编辑器。
  • Simplenote: Markdown 编辑器。
  • Obsidian: Markdown 管理，部分收费。
  • MarkDownEditor: Markdown 编辑器。
  • Mou: Markdown 编辑器。
  • Milkdown: : Markdown 编辑器，WYSIWYG。
  • GitNote: Markdown 编辑器。
  • Yank Note: Markdown 编辑器，作者的原文为什么要自己写笔记软件？。最近发现更换主题都需要收费，改为不推荐。

其他类型的笔记软件:

  • TextBundle: Markdwon 的图片储存格式。
  • Mempad: 树状笔记软件。
  • eDiary: 电子日记本。

  • Simplenote
  • Minipad 2
  • Dynalist
  • Rightnote
  • Workflowy
  • Logseq: 大纲双链式的笔记软件，支持 Markdown。

此外，还有一些 Wiki 类程序:

  • Zim: Wiki 类软件。
  • Kiwix: Wikipedia 本地版，也有浏览器插件见上方。
  • TiddlyWiki: 使用单一的 Html 文件实现笔记管理。我曾当作玩具玩过很久。
  • Stroll: TiddlyWiki 衍生版，实现了双向链功能，即 Roam-like。
  • TiddlyMap: TiddlyWiki 衍生版，实现了思维导图模式。
  • Notebook: TiddlyWiki 的 Notebook 主题模式。

2.5 文本编辑器

  • Brackets: 前端编辑器。Adobe 停止了支持，开发者鼓励迁移到Visual Studio Code。
  • Notepad++: 文本编辑器，作者将政治倾向代入到开源软件中，不推荐。
  • Notepad3: 速度很快，感觉很不错。VScode 没这么快。
  • Sublime Text: 文本编辑器，收费但可以无限试用，不过偶尔会跳出购买窗口。
  • Visual Studio Code: 微软推出的编辑器，有一统江湖的趋势。
  • Kate: 文本编辑器。
  • Notepads: 轻量文本编辑器，是自带记事本的强化版。在 Windows Store 可直接下载。
  • Atom
  • Emacs
  • AkelPad
  • Org Mode: Emacs 插件。

  • FFmpeg: 神器。
  • QuickCut: 视频处理，FFmpeg GUI 界面。
  • HandBrake: 视频压缩与转换，FFmpeg GUI 界面。
  • Axiom: FFmpeg GUI 界面。
  • ShanaEncoder: 视频制作、转码、封装。
  • Natron: 视频特效合成软件。
  • 小丸工具箱: 视频处理。

视频剪辑:

  • Premiere: 即 Pr，收费。
  • ShotCut: Premiere 的开源替代。
  • OpenShot: 视频剪辑。
  • Kdenlive: 视频剪辑。
  • Flowblade: 视频剪辑，仅限 Linux。
  • Avidemux: 视频剪辑。
  • Olive: 视频剪辑。
  • Synfig: 动画制作软件。

一些音频软件:

  • foobar: 音频播放器。
  • MusicBee: 音频播放器。
  • Listen1: 音频播放器。
  • Audacious: 音频播放器。
  • Clementine: 音频播放器。
  • lx-music-desktop: 洛雪音乐助手桌面版。

12 系统检测、优化与管理

一些空间清理和软件卸载程序:

  • CCleaner: 电脑清理，启动项管理，注册表和痕迹清除等，收费。
  • Geek Uninstaller: 卸载软件，专业版收费。
  • HiBit Uninstaller: 卸载软件。
  • IObit Uninstaller: 卸载软件，部分收费。
  • Bulk Crap Uninstaller: 卸载软件，很强大，最推荐。
  • Dism++: 电脑清理等多种功能，作者放弃官网了，换成 Github 链接。
  • WDControl: Windows Defender 状态设置工具。
  • XTR Toolbox: 卸载软件，多年未更新。
  • WindowsAppBoss: 卸载 Windows 自带程序，多年未更新。
  • HDCleaner: 硬盘清理工具，收费。
  • Duplicate Cleaner: 重复文件检测与清理，收费。
  • MemReduct: 内存清理。
  • SpaceSniffer: 磁盘空间分析。
  • UninstallTool: 软件卸载，启动项管理等，部分收费。
  • BleachBit: 清除注册表。

一些软硬件检测和管理软件:

  • HDTune
  • CPU-Z: CPU 等硬件检测。
  • GPU-Z: GPU 检测。
  • CUDA-Z: GPU 检测，相当于 GPU-Z 的开源版本。
  • Memtest: 内存检测。
  • Memtest86+: 内存检测。
  • TrafficMonitor: 国内网友制作。精致小巧，用于监测内存，网络，CPU 等。
  • Process Monitor: 微软开发的进程监视器。
  • AIDA64: 硬件检测，功能不错，但收费。
  • DiskGenius: 磁盘分区，数据恢复软件，收费，免费版提供部分功能。
  • TestDisk: 数据恢复。
  • PhotoRec: 数据恢复。
  • Windows File Recovery: 微软开发的数据恢复软件，无 GUI。
  • GParted: 硬盘分区与管理软件。
  • 图拉丁吧工具箱: 大量硬件检测软件集合，整合了上述各类工具，非常好用。
  • Hashcat: 密码破解器。
