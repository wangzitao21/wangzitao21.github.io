Hugo 安装 Windows 版本的 hugo 安装比较简单，直接在官网下载 hugo_extended 后，将其解压，然后配置环境变量即可。 Linux (Debian) 系统的 Hugo 仓库版本有点老，故采用以下命令安装: # 链接需要更改 wget https://github.com/gohugoio/hugo/releases/download/v0.140.2/hugo_0.140.2_linux-amd64.deb # 解压 sudo dpkg -i hugo_0.140.2_linux-amd64.deb 无论是 Windows 还是 Linux，配置完成均可采用以下命令检查是否安装成功: hugo version 本地配置 初次部署步骤如下: 先在 Git 命令里操作，首先移动到需要建博客的指定位置然后新建站点。 cd /e/onedrive # blog 名可以更改 hugo new site blog --format yaml 进入主题文件夹下载安装主题。该主题具体配置见 PaperMod。 cd /e/onedrive/blog git clone https://github.com/adityatelange/hugo-PaperMod themes/PaperMod --depth=1 在 hugo.yaml 配置文件中添加主题名: baseURL: https://example.org/ languageCode: en-us title: My New Hugo Site theme: ["PaperMod"] 此时已经安装完成了，可以测试一下: cd /e/onedrive/blog # 如果是在服务器上可以修改 IP hugo server --bind=0.0.0.0 --baseURL=http://127.0.0.1 --port=1313 非初次部署是指已经有 hugo 博客文件夹了。部署步骤基本一致，从上述第 2 步开始即可。 ...
