How to Refresh Environment Variables Without Restarting Windows

5/19/2024 ¬∑ Zitao Wang | Translations:
  • üåê ÂàáÊç¢‰∏∫‰∏≠Êñá

In the Windows system, sometimes when adding environment variables for some executable files, such as ffmpeg, modflow, etc., it usually requires a system restart to take effect.

However, restarting often requires closing many ongoing tasks, which is very troublesome.

The following steps can refresh environment variables without restarting the system:

  1. Add the path of the program you need to add to the environment variables.
  2. Open CMD and enter echo %Path% to view the current environment variables.
  3. Enter set PATH=C:.
  4. Close CMD and reopen it. Entering echo %Path% again will show that the environment variables have been updated.
